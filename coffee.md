## Usage / Methods

- xR = Ratio, example 16.7 -> 12g/200ml
- A = AeroPress

  - i = inverted
  - xm = time
  - b = +bloom/stir (usually 30s)
- B = Bialetti / Moka pot
- E = Espresso / 9bar
- P = E.S.E. pads
